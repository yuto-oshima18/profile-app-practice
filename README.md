# profile-app-practice

## 概要
COACHTECH Laravel基礎 「9-1-8: Laravel基礎ハンズオン」にて作成

## 使用技術
- PHP 8.5.8
- Laravel 10.50.2
- Blade テンプレート

## 学んだこと
- リクエストからビューで表示するまでの一連の流れを体感
- コントローラーの作成方法(sail artisan make:controller ~)
- 連想配列に格納された値をビューに送る手段

## 動作確認
- ①sailコマンドによりバックグラウンドで起動(sail up -d)
- ②sailコマンドによるコンテナ状態確認(sail ps)
- ③http://localhost/profileにアクセス
- ④画面を確認し、表示された内容が正しいことを確認
